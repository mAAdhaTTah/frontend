# jamesdigioia.com

**My little web home** — a personal site for writing, reading, code, and talks.

---

## What's Here

### [Writing](https://jamesdigioia.com/writing/)

Essays, concepts, and notes from my second brain. A mix of technical deep-dives, career reflections, and the occasional travel journal. Things I've worked through and wanted to put somewhere permanent.

### [Reading](https://jamesdigioia.com/reading/)

A running feed of things I've read, watched, and found worth bookmarking — tech, politics, poker strategy, and whatever else crossed my path recently.

### [Code](https://jamesdigioia.com/gistpens/)

Practical snippets and examples. Stuff I've reached for more than once, or that was tricky enough to be worth writing down.

### [Talks](https://jamesdigioia.com/talks/)

Slide decks from conference talks I've given, mostly around JavaScript, React, and frontend engineering.

### [Resume](https://jamesdigioia.com/resume/)

Where I've worked and what I've built.
